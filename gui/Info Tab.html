Four-Switch Buck-Boost Development Board - Protocol

Length:

Only the lowest 15 bits of the length-word can be used for describing the length of the data. The highest bit of the length-word is being used for determining CRC-usage.

CRC:

The highest bit of the length-word in all data transmissions determines if CRC is being used or not.
0: CRC has to be 0x0000. for receiving, the data package needs to be checked for CRC = 0x0000.
1: CRC-16 with start polynom 0xA001 is being used.

Status message data words:

0: Vdd
1: Vout
2: Iout
3: Iin
4: AN15
5: Vaux
6: Power controller state
7: Collective faults
8: Gui flags
9: Power controller flags
10: Duty cycle buck pwm
11: Duty cycle boost pwm
12: Current control result
